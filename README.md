# nest-crud-ws
Nest crud api &amp; websocket app.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
Goto url domain.com/api for accessing swagger documentation.
